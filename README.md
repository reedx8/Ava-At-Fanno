# Real Estate Website (Ava)

Ava at Fanno Creek real estate landing page

## To Run

1. Clone repo
2. Run in project directory:
    1. `npm install`
    2. `npm run dev`
3. Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
